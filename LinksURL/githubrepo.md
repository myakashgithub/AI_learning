10 GitHub repos that teach you everything you need to build AI Agents

# 1. Machine Learning for Beginners 

Covers core ML concepts such as supervised learning, model evaluation, data preprocessing, and feature engineering, with structured lessons and hands-on notebooks.

🔗 https://lnkd.in/dBj3BAEY

# 2. Hands-On Large Language Models

Complete set of runnable code examples from the book, covering foundational concepts to advanced LLM fine-tuning.

🔗 https://lnkd.in/dxaVF86w

# 3. LLM Course

A public repo that provides a multi-part course on LLMs, including fundamentals, building LLM-powered apps, and deployment techniques with hands-on notebooks.

🔗 https://lnkd.in/diZgGACG

# 4. Prompt Engineering Guide

Curated repository of notes, research links, guides, and other resources focused on prompt design techniques, RAG strategies, context engineering, and related LLM practices. 

🔗 https://lnkd.in/dBUCsVJ8

# 5. Designing Machine Learning Systems

A companion repository that aggregates summaries and notes from the Designing Machine Learning Systems book, focusing on architecture, scaling, monitoring, and production principles.

🔗 https://lnkd.in/dEx8sQJK

# 6. Made With ML

A collection of end-to-end machine learning projects and examples showing how ML systems are designed, trained, deployed, and iterated in realistic scenarios. 

🔗 https://lnkd.in/d2dMACMj

# 7. AI Agents for Beginners

A structured, lesson-based course repository from Microsoft covering fundamental concepts and code examples for building AI agents, including task setup, basic agent loops, and tool integration.

🔗 https://lnkd.in/dHvTmJnv

# 8. GenAI Agents

Contains tutorials and open-source code demonstrating generative AI agent workflows and implementations, from simpler agent designs to more complex interactive systems. 

🔗 https://lnkd.in/dEt72MEy

# 9. Hands-On AI Engineering

A showcase of practical examples and code for building AI-powered applications and agent-like systems using LLMs and related tooling. 

🔗 https://lnkd.in/dgQtRyk7

# 10. Awesome Generative AI Guide

A curated index of generative AI resources, including research papers, interview prep material, free courses, notebooks, and tool listings, updated regularly.
